About This Dashboard

This dashboard tracks safety mechanism implementations across 14 LLM providers by analysing 40 source documents (system cards, technical reports, model cards, and documentation).

Methodology: Safety techniques are detected using a multi-stage pipeline:

  1. Semantic Retrieval: Identifies candidate text passages using sentence embeddings (all-mpnet-base-v2)
  2. Entailment Verification: Validates technique presence using cross-encoder verification (nli-deberta-v3-small)
  3. LLM Review: Claude reviews findings in context, confirming matches and flagging gaps
  4. Quality Filtering: Removes glossary definitions, "future work" mentions, and other false positives

Confidence Levels: High = strong implementation evidence with specific technical language; Medium = clear mentions; Low = indirect or weak references.
