About This Dashboard

This dashboard tracks safety mechanism implementations across 15 major LLM providers by analyzing 40 source documents (system cards, technical reports, model cards, and documentation).

What This Shows:

  • Coverage Heatmap: Which providers implement which safety techniques based on their public documentation
  • Risk Area Coverage: How well different categories of AI risk are addressed across the industry

Methodology: Safety techniques are detected using a two-stage NLU pipeline:

  1. Semantic Retrieval: Identifies candidate text passages using sentence embeddings (all-mpnet-base-v2)
  2. Entailment Verification: Validates technique presence using cross-encoder verification (nli-deberta-v3-small)
  3. Quality Filtering: Removes glossary definitions, "future work" mentions, and other false positives

Confidence Levels: High confidence indicates strong implementation evidence with specific technical language; Medium indicates clear mentions; Low indicates indirect or weak references.
